# Maintenance – Service Request

The Service Request page allows you to create, monitor, and manage maintenance requests for vending machines.

---

## 1. Page Overview
When you open the Service Request page, the following option is available at the top-right corner:

### 1.1 Request a Service Button
The Request a Service button allows you to create a new maintenance request for a vending machine. You can provide issue details and assign the request for resolution.

---

## 2. Service Status Overview
Below the top section, three summary cards provide a quick overview of service request status:

### 2.1 Open
Displays the total number of service requests that are newly created and pending action.

### 2.2 In Progress
Shows the number of service requests currently being handled.

### 2.3 Completed
Indicates the number of service requests that have been successfully resolved.

These summary cards help you quickly understand the maintenance workload and current status distribution.

---

## 3. Search & Filter Options
Below the status cards:

### 3.1 Search Option
On the left side, a Search button allows you to quickly find specific service requests by location, machine number, or issue details.

### 3.2 Filter Option
On the right side, the Filter button allows you to refine service requests based on selected criteria such as status or issue type.

---

## 4. Service Request Details
The Service Request Details table displays structured information about each maintenance request.

The table includes the following columns: Date, Location, Machine No., Issue Category, Issue Type, Description, Assigned To, Status, Comments, and Action.

This section allows you to:

- Review reported machine issues  
- Monitor request progress (Open, In Progress, Completed)  
- View assigned personnel  
- Check issue descriptions and comments  
- Manage requests using the Action option
