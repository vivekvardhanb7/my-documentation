# Cloud System – Home Page User Guide

## Home
When you log in to the Cloud System, the Home page opens by default. It provides a quick overview of sales performance, machine status, inventory levels, maintenance updates and machine locations in one place.

---

## 1. Top Navigation Bar
At the top of the Home page, you will see the main navigation bar. This section allows you to manage system settings and access important options.

### 1.1 Sidebar Icon
Opens or closes the side navigation menu.

### 1.2 Theme Icon (Sun/Moon)
Switches between Light Mode and Dark Mode.

### 1.3 Language Switch
Allows you to change the system language. Five languages are available.

### 1.4 Notification Icon
Displays new alerts and system notifications.

### 1.5 Profile Dropdown
Provides access to user profile options:

#### 1.5.1 Profile
View and manage your profile information.

#### 1.5.2 Settings
Update language preference and time zone

#### 1.5.3 Password
Change your password using Current Password, New Password, and Confirm Password fields.

#### 1.5.4 Account
Update company details such as Company Name, Phone Number, Email, Address, and Tax information.

#### 1.5.5 Log Out
Securely sign out from the system.

---

## 2. Side Navigation Menu
On the left side of the screen, you will see the Side Navigation Menu. This menu allows you to quickly access different sections of the Cloud System.

### 2.1 Home
Opens the main dashboard where you can view overall system performance, sales summary, machine status, and key insights.

### 2.2 View
Allows you to monitor and review system data such as sales analytics and transaction history.

### 2.3 Manage
Provides options to manage vending machines, inventory, advertisements, discounts, coupons, users, and system configurations.

### 2.4 Tax
Opens the Tax Management page where you can view tax details, track tax deadlines, and manage tax-related information.

### 2.5 Maintenance
Opens the Maintenance page where you can track service requests, scheduled maintenance, and machine repair status.

### 2.6 Help
Provides access to help documentation and user guidance to better understand and operate the system.

---

## 3. Today’s Sales
This section shows the total sales generated for the current day, categorized by payment method such as Membership Card and Credit Card. It helps you quickly understand daily revenue performance. An arrow icon is available in the top-right corner. Clicking this arrow opens the detailed Sales Analytics page.

---

## 4. Sales Analysis
Displays sales performance in graphical format. You can select different time periods such as Day, Week, Month, or Year to analyze trends. Click the arrow icon in the top-right corner to navigate directly to the Sales Analytics page.

---

## 5. Inventory Level
Shows the current stock levels for each vending machine. It helps you identify whether stock is sufficient, low, or requires restocking. Click the arrow icon in the top-right corner to open the Vending Machines page for detailed inventory management.

---

## 6. Machine Status
The Machine Status section provides an overview of each vending machine’s operational condition. It includes Active/Inactive status, temperature range, location, and overall condition (Normal/Abnormal). Click the arrow icon in the top-right corner to open the Vending Machines page for more details.

---

## 7. Map View
The Map displays the locations of all vending machines. You can search by address or machine number to quickly locate a specific machine.
