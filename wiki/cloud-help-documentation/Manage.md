# Manage

## Vending Machines
The Vending Machines page under the Manage section allows you to monitor, filter, and manage all vending machines from a centralized dashboard.

---

## 1. Vending Machines Overview
When you open the Vending Machines page, the page title “Vending Machines” appears at the top.

On the right side, the following options are available:

### 1.1 Search Option
Allows you to quickly search for a specific vending machine using keywords.

### 1.2 Filter Option
Clicking the Filter button opens a dropdown with the following options: All, Active, Inactive, Normal, and Abnormal. Selecting any option filters the machines based on status or condition.

---

## 2. Machine Information Cards
Each vending machine is displayed in an individual card format containing detailed information.

Each machine card includes:

- Location Name (e.g., Berlin)
- Machine Identifier (e.g., S Messe Sued - 14012026)
- Status (Active / Inactive)
- Condition (Normal / Abnormal)
- Live Temperature (e.g., 9°C)
- Inventory Level Percentage (e.g., 38.96%)
- Restock Indicator
- Export Option

This section helps you quickly monitor machine health, stock levels, and operational status.

### 2.1 Machine Details Navigation
On the top-right corner of each machine card, there is an arrow icon. Clicking this arrow opens a detailed machine information page.

---

## Machine Detailed View
When you click the arrow icon, a new page opens with two tabs at the top:

- Machine
- Aisle

---

## 3. Machine Tab
The Machine tab displays complete machine information.

### 3.1 Machine Information
This section includes Machine Type, Status, Machine Number, Location, Street Name, Inventory Levels, Condition, and Live Temperature.

### 3.2 Temperature Settings
Below the machine information, there is a Temperature Settings section where you can adjust temperature configurations. After making changes, click the Save button to apply updates.

### 3.3 Previous Temperature Conditions
This section displays historical temperature records for the machine. On the right side, an Export button is available to download temperature data, along with a Calendar icon to filter temperature history by selected dates (Daily or Continuous view with Apply option).

---

## 4. Aisle Tab
When you click the Aisle tab, detailed product information is displayed.

### 4.1 Aisle Information
This section shows the Number of Items and Pricing details for each aisle inside the vending machine.

The layout and structure are similar to the machine overview cards shown on the main Vending Machines page.

---

# Inventory
The Inventory page allows you to manage product categories, add new products, and monitor item pricing and availability.

---

## 1. Inventory Overview
When you open the Inventory page, the title “Inventory” appears at the top left.

On the top-right side, the following options are available:

### 1.1 Categories Button
Clicking the Categories button opens a list of available product categories. Each category displays the number of products available within it on the right side.

### 1.2 Add Product Button
Clicking the Add Product button opens a product creation form where you can upload the product image and enter product details. After entering the required information, you can click Save to add the product or Cancel to discard changes.

---

## 2. Category Navigation Bar
Below the top section, a horizontal navigation bar is displayed.

This includes category labels such as Week 8 2026, Meat, Rice Bowl, Moroccan, and other available categories. Clicking on any category filters the products displayed below based on the selected category.

This navigation bar allows quick switching between product groups.

---

## 3. Product Listing Section
Below the category navigation bar, all items are displayed with product images and pricing details.

At the top of this section:

### 3.1 Search Option
A Search button is available on the left side to quickly find specific products.

### 3.2 Sort Option
On the right side, the Sort button provides a dropdown with options All, Low to High, and High to Low, allowing you to sort products based on price.

### 3.3 Filter Option
Next to the Sort button, the Filter button provides a dropdown containing All and individual item names, allowing you to filter products by specific items.

---

## 4. Product Details Display
Below the filter and sort options, products are displayed in a structured layout showing Item Image and Price.

This section allows you to easily review available inventory items and monitor product pricing.

---

# Advertisement
The Advertisement page allows you to create and manage promotional content displayed on vending machines.

---

## 1. Advertisement Overview
When you open the Advertisement page, you will find the advertisement management options clearly organized.

On the top-right side, the following option is available:

### 1.1 New Advertisement Button
Clicking the New Advertisement button allows you to create and configure a new advertisement. You can define the advertisement details and save it for activation.

---

## 2. Advertisement Sections
Below the top section, advertisements are categorized into two sections:

### 2.1 Live Advertisements
This section displays all advertisements that are currently active and running on vending machines.

### 2.2 Saved Advertisements
This section contains advertisements that have been created and saved but are not currently live.

---

# Manage – Discount
The Discount page allows you to create, manage, and monitor discount campaigns applied to products or transactions.

---

## 1. Discount Overview
When you open the Discount page, discount management options are clearly displayed.

On the top-right side:

### 1.1 New Discount Button
Clicking the New Discount button allows you to create a new discount offer. You can configure discount details and save them for activation.

---

## 2. Search, Filter & Export Options
Below the top section, management tools are available to monitor discount-related records.

### 2.1 Search Option
On the left side, a Search button allows you to quickly find specific discount records or related transactions.

### 2.2 Filter Option
On the right side, the Filter button allows you to refine results based on selected criteria.

### 2.3 Export Option
Next to the Filter button, the Export option allows you to download discount-related data for reporting and record-keeping purposes.

---

## 3. Discount Details
Below the search and filter section, the Discount Details table displays structured information about each discount.

The table includes the following columns: Name, Machines, Start Date, End Date, Status, Percentage, and Action.

This section helps you monitor active and expired discounts, check the machines where discounts are applied, review discount duration, and manage actions such as editing or updating discount configurations.

---

# Coupons
The Coupons page allows you to generate, manage, and monitor coupon campaigns used across vending machines.

---

## 1. Coupons Overview
When you open the Coupons page, coupon management options are displayed clearly.

### 1.1 Generate Coupon Button
On the top-right side, the Generate Coupon button allows you to create new coupon codes. You can configure coupon details such as value, applicable machines, and validity period before activating them.

---

## 2. Search, Filter & Export Options
Below the top section, management tools are available to organize and review coupon records.

### 2.1 Search Option
On the left side, the Search button allows you to quickly find specific coupons by name or related details.

### 2.2 Filter Option
On the right side, the Filter button helps you refine coupon records based on selected criteria.

### 2.3 Export Option
Next to the Filter button, the Export option allows you to download coupon data for reporting and record-keeping purposes.

---

## 3. Coupon Details
Below the search and filter section, the Coupon Details table displays structured information about each coupon.

The table includes the following columns: Name, Coupons, Each Coupon Price, Machines, Start Date, End Date, Usage, Status, and Action.

This section helps you monitor coupon quantity, pricing, machine assignment, validity duration, usage tracking, activation status, and manage actions such as editing or updating coupon configurations.

---

# Users
The Users page allows you to manage system users, assign roles, and control access permissions within the cloud system.

---

## 1. Users Overview
When you open the Users page, user management options are clearly displayed.

### 1.1 New User Button
On the top-right side, the New User button allows you to create and add a new user to the system. You can enter user details and assign an appropriate role based on access requirements.

---

## 2. Search Option
Below the top section, a Search button is available to quickly find users by name, email address, or other details.

---

## 3. User Details
Below the search section, the User Details table displays structured information about each registered user.

The table includes the following columns: Name, Email, Phone Number, Address, Role, and Action.

This section allows you to:

- View user contact information  
- Identify assigned roles such as Super Admin, Admin, Employee, or Support  
- Manage user accounts using the Action option
